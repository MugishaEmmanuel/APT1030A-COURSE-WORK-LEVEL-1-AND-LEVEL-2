# eCitizen & SACCO Systems — Syntax, Semantics, and Program Correctness

## Overview
This repository contains exercises in **Python, Java, and C++** demonstrating **syntax, semantics, and program correctness**.  

Two exercises are included:  

1. **Level 1 — eCitizen Login Validation System**: Verifies username and password input.  
2. **Level 2 — SACCO Member Financial Record**: Calculates a member’s total savings over 6 months.  

---

## Features
- Programs in **Python, Java, and C++** (Level 1)  
- Programs in **Python and Java** (Level 2)  
- Demonstrates:
  - Syntax errors  
  - Logical errors  
  - Type handling (dynamic vs static typing)  

---

## Key Takeaways
- **Static typing** (Java, C++) catches type errors at compile-time, making financial systems safer.  
- **Dynamic typing** (Python) allows flexible and faster development, ideal for prototyping.
